# Smart Waste Reporting Platform

An AI-powered prototype that enables citizens to report urban waste issues using images and location data, and provides authorities with a dashboard to prioritize cleanup efficiently.

## Features
- Waste image upload
- AI-based waste type and severity detection (prototype)
- Location tagging using map interface
- Authority dashboard with severity-based visualization

## Tech Stack
- Frontend: HTML, CSS, JavaScript / React
- Backend: Python (Flask / FastAPI)
- AI: Image classification (prototype logic)
- Maps: Leaflet / Google Maps

## Project Status
This project is a hackathon MVP focused on core functionality and scalability.
